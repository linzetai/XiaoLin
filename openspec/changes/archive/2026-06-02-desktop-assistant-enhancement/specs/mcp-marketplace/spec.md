## Overview

Curated MCP server discovery, installation, and lifecycle management GUI.

## Requirements

- Settings page "MCP 市场" tab with curated server list
- Local JSON registry bundled with app (no online marketplace server in Phase 1)
- One-click install: downloads server binary + writes mcpServers config
- Server status display: running, stopped, error
- Uninstall: removes config entry + optional binary cleanup
- Categories: productivity, development, communication, data
