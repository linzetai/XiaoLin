# UI 体验优化第一轮

## 概述

对 xiaolin-app 前端进行一轮 UX 打磨，修复影响日常使用的交互问题和视觉不一致。

## 动机

1. 侧边栏"折叠"后仍占 52px 宽度，并非真正隐藏
2. 会话结束后，工具调用和文本不再交错显示，工具被集中折叠在顶部
3. 模型选择器仅在组件挂载时拉取一次模型列表，设置中新增/修改模型后不会同步
4. 各页面设计细节风格不统一

## 范围

- `crates/xiaolin-app/src/` 前端代码
- 不涉及后端逻辑改动（问题 2 采用纯前端方案）
- 不涉及新功能，纯 polish

## 目标

- 侧边栏折叠时完全隐藏（0 宽度），聊天区占满
- 已完成会话的消息保持文本与工具调用的原始交错顺序
- 模型选择器能实时反映最新的模型配置
- 统一 input/button/card 等基础组件的视觉表现
